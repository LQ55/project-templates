### 项目结构
```shell
├── src                                    // 项目源码，包含程序源码和测试源码
│   ├── api                                // 抽取出的api请求
│   ├── assets                             // 存放静态资源的文件夹
│   ├── components                         // 项目组件的文件夹
│   ├── router                             // 路由
│   ├—— store                              // 状态管理
│   ├── test                               // 测试源码存放目录
├── .gitignore                             // 忽略提交文件配置
├── package.json                           // package.json
├── README.md                              // 项目简介与说明
```
